## 🚀 Quick Start

Each project is a standalone Python application. To get started:

```bash
# Navigate to a project
cd assistant

# Install dependencies (using uv - recommended)
uv sync

# Configure environment
cp .env.example .env
# Edit .env and add your GEMINI_API_KEY

# Run the project
uv run python main.py
```

## 🔑 Requirements

- **Python**: 3.13+
- **Package Manager**: [uv](https://github.com/astral-sh/uv) (recommended) or pip
- **API Key**: Gemini API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
